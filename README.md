## Description

CloudWalk test.

The following project provides an API who extract and process a Quake game log
returning 2 lists, one with the best ranked players and other with the weapons kills rank

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
